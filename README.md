# Tampere Design System

Design system for the City of Tampere

Tampere Design System (TREDS) is built by the City of Tampere and is meant for building accessible solutions with a great user experience.

This react component library is based on [Mantine](https://mantine.dev/)

## Installation

```bash
npm install
```

## Development in Storybook

```bash
npm run storybook
```

## Eslint and Prettier

```bash
npm run eslint
```

```bash
npm run format
```

## Build with rollup

```bash
npm run build
```
